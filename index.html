Closed-Loop Smart Servo Control System

An advanced, high-precision position controller driven by an ESP32 microcontroller, utilizing real-time I2C feedback from an AS5600 encoder to drive a geared DC motor.

How It Works

ServoFlow functions as a high-frequency feedback loop that reads sensor angles and adjusts voltages instantly to stay locked on the target.

1

Setpoint Command

The user selects a target angle (0-360°) via the React Native dashboard or the physical rotary encoder.

2

PID Processing

The ESP32 runs a 100Hz PID loop, computing corrections based on error, deadband offset, and filtered derivative speed.

3

Motor Actuation

The HW-095 (L298N) H-bridge driver receives PWM signals and outputs drive current to actuate the TT DC motor.

4

Encoder Feedback

The AS5600 absolute magnetic encoder reads the output shaft's physical angle at 12-bit resolution and feeds it back via I2C.

Hardware Components

The core hardware components configured inside the ServoFlow smart actuator loop.

Controller

ESP32 NodeMCU

Executes the 100Hz PID calculations, processes encoder interrupts, displays menus, and runs the WebSocket telemetry server.

Feedback Sensor

AS5600 Magnetic Encoder

Provides 12-bit absolute angle measurement (4096 positions) via I2C. Avoids mechanical wear and wear-out over time.

Actuation Driver

HW-095 L298N Driver

A dual-channel H-Bridge motor driver that translates logic level PWM direction commands into high-power motor drive current.

Actuator

TT Geared DC Motor

Provides robust torque and rotation. Controlled via deadband-compensated proportional-integral-derivative inputs.

Display

0.96" SSD1306 OLED

Displays real-time actual angles, active connection indicators, current PID values, and diagnostic warnings.

Physical Input

KY-040 Rotary Encoder

Enables local menu navigation, digit-by-digit tuning, target angle adjustment, and calibration commands.

Setup Instructions

Follow these structured steps to flash the firmware, run the dashboard, and configure the system.

1

Hardware Wiring

Wire the components according to the pinout mapping:

  • Encoder / Display: SDA to Pin 21, SCL to Pin 22 (shared I2C bus).
  • Motor: IN1 to Pin 25, IN2 to Pin 26 (PWM driving).
  • Rotary Encoder: CLK to Pin 18, DT to Pin 19, SW to Pin 5.
  • Bridge all Ground connections (ESP32 and Motor Battery pack).
2

Flash Firmware

Upload the firmware to your ESP32:

  • Open ESP32/ServoFlow/ServoFlow.ino in the Arduino IDE.
  • Install dependencies: AS5600, Adafruit SSD1306, WebSockets, and ArduinoJson.
  • Configure board to ESP32 Dev Module and click upload.
3

Deploy App Dashboard

Run the mobile monitoring client:

  • Run npm install in the root directory.
  • Run npm run dev to boot up the Expo server.
  • Connect your phone to the ESP32 hot-spot WiFi: SSID: ServoFlow, Pass: sapumal2003.
  • Launch Expo Go and view live telemetry!
